# NeuroNetix Project
AI-based Brain-Computer Interface (BCI) system using EEG signal acquisition, embedded hardware, and real-time brainwave analysis.

## Overview
NeuroNetix is designed to monitor and analyze brain activity using EEG sensors and AI-based processing techniques. The system captures EEG 
signals from the human brain, processes them using embedded hardware, and visualizes the results through a real-time monitoring dashboard.
The project focuses on detecting mental states such as relaxation, concentration, and stress using EEG signal analysis .

## Features
- Real-time EEG signal acquisition
- Brainwave monitoring and visualization
- Mental state detection
- Stress level analysis
- EEG signal filtering and processing
- Session summary and report generation
- AI-based brain activity analysis

## Hardware Components
- BioAmp EXG Pill
- EEG Electrodes
- Arduino Microcontroller
- I2C Module
- External Display
- Speaker Module

## Software Components
- Python
- Arduino IDE
- VS Code
- Real-time EEG Dashboard

## System Architecture
1. EEG electrodes capture brain signals from the scalp
2. BioAmp EXG Pill amplifies weak EEG signals
3. Arduino converts analog signals into digital data
4. Signals are filtered and processed to remove noise
5. Features are extracted from the EEG signals
6. Machine Learning models classify mental states
7. Results are displayed on the dashboard in real time
8. Session data is stored and summarized

## Modules
### Module 1 – Patient Details Acquisition
Collects patient information before EEG monitoring.

### Module 2 – Patient Verification
Verifies entered patient information.

### Module 3 – EEG Device Connection
Connects the system with the BioAmp EXG Pill EEG device.

### Module 4 – Real-Time EEG Monitoring Dashboard
Displays EEG waveforms, stress levels, brain activity, and signal analysis.

### Module 5 – EEG Session Summary and Analysis
Generates session reports and AI-based insights.
